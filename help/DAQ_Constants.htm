DAQ Toolbox Functions

DAQ_Constants - DAQ

Calling Sequence

[ value ] = DAQ ( name )

Parameters

  • name : string
  • value : integer

Description

DAQ("NAME") returns the value of a predefined constant DAQmx_NAME.

Examples

   // value of DAQmx_Val_GroupByChannel
   value = DAQ("Val_GroupByChannel");
