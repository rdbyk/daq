DAQ Toolbox Functions

DAQ_Task_Configuration_and_Control - DAQ_CreateTask, DAQ_LoadTask, DAQ_StartTask, DAQ_StopTask, DAQ_ClearTask, DAQ_IsTaskDone, DAQ_ResetDevice

Calling Sequence

[ taskHandle, status ] = DAQ_CreateTask ( taskName )

[ taskHandle, status ] = DAQ_LoadTask ( taskName )

[ status ] = DAQ_StartTask ( taskHandle )

[ status ] = DAQ_StopTask ( taskHandle )

[ status ] = DAQ_ClearTask ( taskHandle )

[ isTaskDone, status ] = DAQ_IsTaskDone ( taskHandle )

[ status ] = DAQ_ResetDevice ( deviceName )

Parameters

  • taskName : string
  • taskHandle : mlist (type "daq")
  • isTaskDone : boolean
  • deviceName : string
  • status : integer

Description

DAQ_FunctionName provides just an interface to DAQmxBaseFunctionName.

For more detailed information about the interfaced functions, please refer to NI-DAQmx Base 2.x C Function Reference Help.
