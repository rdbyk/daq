Calling Sequence

[ status ] = DAQ_DisableStartTrig ( taskHandle )

[ status ] = DAQ_CfgDigEdgeStartTrig ( taskHandle, triggerSource, triggerEdge )

[ status ] = DAQ_CfgAnlgEdgeStartTrig ( taskHandle, triggerSource, triggerSlope, triggerLevel )

[ status ] = DAQ_DisableRefTrig ( taskHandle )

[ status ] = DAQ_CfgDigEdgeRefTrig ( taskHandle, triggerSource, triggerEdge, pretriggerSamples )

[ status ] = DAQ_CfgAnlgEdgeRefTrig ( taskHandle, triggerSource, triggerSlope, triggerLevel, pretriggerSamples )

Parameters

  • taskHandle : mlist (type "daq")
  • triggerSource : string
  • triggerEdge : integer
  • triggerSlope : integer
  • triggerLevel : double
  • pretriggerSamples : integer
  • status : integer

Description

DAQ_FunctionName provides just an interface to DAQmxBaseFunctionName.

For more detailed information about the interfaced functions, please refer to NI-DAQmx Base 2.x C Function Reference Help.
